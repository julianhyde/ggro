Help For the GGRO Data Entry / Admin Program

What are the admin functions? 

SetDate - Use this to change the date. Note that it won't stop you from entering a date that isn't in the database.

Set Database - Toggle between the Entry database and the Permanent Database. You can edit the data in either.

Commit to Main - Copy the entire contents of the entry database into the permanent database. The entry database will be emptied in the process. The intent is to clean the data in the entry database, as needed, and commit to the main database.


How do I enter the data?

--The program is designed to look pretty much like the field form. Copy the non-zero values for each species for each hour into the place where they appear in the program. It will calculate the totals automatically. You can use the mouse, enter key, tabs, etc, as you like. 

What should I enter on the Dark Morph tab? This doesn't look like the field form.

--Enter the species under the appropriate hour column. (There are no totals for dark morphs).

Where do I enter the date?

--The date is set automatically to "today".

How do I save the data?

--Select "Save Data" from the File menu.

How do I exit the program?

--Select "Exit" from the File menu.

I closed the program but noticed that I forgot to enter some data. Now what do I do?

--Simply start the program again, and finish entering the data.

When I re-opened the program, there are a bunch of zeroes that I didn't enter. Should I be worried? 

--No.

I saved the data, but I made a mistake.  Can I just change it and save it again?

--Yes.

The Save Data menu item is not enabled, so I can't save the data!

--Either no data has been entered, or you already saved it. If you think you haven't saved the data and now can't for some reason, do the following: Enter a zero somewhere it won't make any difference (perhaps "Goshawk" has a zero column today); hit  the Enter key. Now "Save Data" should be enabled.
